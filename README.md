# 1.基础知识

* 代码版本: 1.0.0-Release

* 功能：使用Lombok简化代码

* 包含模块：  
    * LombokRequest：演示使用Lombok简化代码
    * LombokController：演示效果
    
* 启动项目并验证
    * 启动zh-boot
    * 访问 http://localhost:8080/lombok
    * 参数：
    ```json
      {
        "intLombok":3,
        "strLombok":"zhanghan",
        "boleanLombok":true,
        "personLombok":{
          "name":"zhangsan",
          "age":15
        }
      }
    ```
